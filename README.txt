# README

## After cloning the repository

1. Run `npm install` to install dependencies.
2. Set ESLint as a default formatter for javascript files in VSCode.

## Making PR

1. Create a new feature branch (`feature/hyphened-branch-name`).
2. Make your changes.
3. Check for type errors by running `npm run type-check`.
4. Check for linting errors by running `npm run lint`.
5. Create a PR.
